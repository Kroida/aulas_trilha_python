# Amigo Secreto em Python

## Descrição
Este projeto consiste em um programa para realizar o sorteio do Amigo Secreto, desenvolvido em Python. O objetivo é sortear aleatoriamente os participantes, garantindo que ninguém tire a si mesmo e que todos participem de forma justa.

## Conteúdos Estudados
- Estruturação do problema do Amigo Secreto
- Manipulação de listas para armazenar e sortear participantes
- Uso do módulo `random` para embaralhamento e sorteio
- Implementação de verificações para garantir sorteios válidos
- Modularização do código com funções
- Validação de entradas e tratamento de exceções
- Boas práticas de programação e padronização conforme PEP 8

## Organização do Código
1. **Coleta dos participantes:** O programa solicita os nomes dos participantes e armazena em uma lista.
2. **Sorteio:** Utiliza o módulo `random` para embaralhar e sortear os pares, garantindo que ninguém tire a si mesmo.
3. **Exibição dos resultados:** Mostra o resultado do sorteio de forma clara e individual.

## Semântica e Boas Práticas
- Nomes de variáveis claros e descritivos
- Separação de responsabilidades em funções
- Comentários explicativos no código
- Validação das entradas do usuário
- Estruturação e indentação adequadas
- Testes manuais para garantir o funcionamento

## Como Executar
1. Certifique-se de ter o Python instalado (versão 3.x).
2. Execute o arquivo `amigo_secreto.py`:
   ```bash
   python amigo_secreto.py
   ```
3. Siga as instruções no terminal para inserir os participantes e visualizar o sorteio.
